Quick Start

# Clone and install
git clone https://github.com/Yash-Sukhdeve/universal-workflow-system.git
cd universal-workflow-system
./install.sh

# In your project directory
uws init software      # Initialize workflow
uws agent implementer  # Activate an agent
uws sdlc start         # Start SDLC workflow
uws checkpoint create "Feature complete"
uws status             # Check current state
